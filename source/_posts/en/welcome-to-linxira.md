## What is Linxira OS?

**Linxira OS** is a **Debian 13 Trixie**-based Linux distribution for scientific computing, targeting bioinformatics, AI/ML, chemistry, physics, and mathematics.

## Key Features

| Domain | Pre-installed Tools |
|--------|-------------------|
| 🧬 Bioinformatics | BLAST, MUSCLE, MAFFT, IQ-TREE, SAMtools, BCFtools, BEDtools, BioPerl |
| 🤖 AI/ML | Jupyter, NumPy, SciPy, Pandas, Scikit-learn, PyTorch, TensorFlow |
| 📐 Math/Statistics | Octave, R, SymPy, Gnuplot, Maxima |
| 🧪 Chemistry/Molecular | Avogadro, PyMOL, OpenBabel, RDKit |
| ⚛️ Physics/Simulation | Geant4, ROOT, KiCad, FreeCAD, CalculiX |

## Quick Start

1. Download the ISO
2. Write to USB: `dd if=linxira.iso of=/dev/sdX bs=4M status=progress`
3. Boot and install
4. Choose GNOME / KDE / XFCE desktop during installation

## Ecosystem

- **CMYKE** — AI assistant (user-level daemon)
- **N-T-AI** — AI/ML workload manager
- **Based on Debian Trixie** with unattended security updates
